# projeto-front-end-faculdade
Projeto de front-end desenvolvido para a disciplina da faculdade.

Este projeto foi desenvolvido para fins acadêmicos. O sistema inclui página inicial, página de projetos, página de cadastro e estilos personalizados.

Estrutura do Projeto

/
├── index.html
├── projetos.html
├── cadastro.html
├── styles.css
├── main.js
└── imagens/ 

Descrição dos Arquivos

index.html
Página inicial do site, contendo navegação global, seção hero e carrossel de Missão, Visão e Valores.

projetos.html
Página destinada à exibição de projetos e iniciativas, organizada em cards responsivos utilizando grid.

cadastro.html
Página com formulário completo, estruturado por fieldsets, com inputs estilizados, radio buttons, checkboxes e áreas de texto.

styles.css
Arquivo principal de estilização. Inclui:

Reset moderno e base tipográfica

Sistema de variáveis para cores e espaçamentos

Grid responsivo de 12 colunas

Botões, cards, alerts e demais componentes

Estilos completos para formulários

Layouts Hero

Estrutura para modais com a tag dialog

main.js
Arquivo de JavaScript responsável pela interatividade, incluindo:

Controle do carrossel

Abertura e fechamento de modais

Funções auxiliares relacionadas ao comportamento da interface

Tecnologias Utilizadas

HTML5

CSS3 (Grid Layout, variáveis, responsividade, design system)

JavaScript (DOM, lógica do carrossel, interações básicas)
